This page is pre-rendered at build time and served as a static HTML file. Static site generation provides blazing fast loading times and excellent caching.

Benefits of SSG:

  • Extremely fast page loads
  • Lower server costs (no processing per request)
  • Excellent for content that doesn't change frequently
  • Can be deployed to CDNs for global distribution
